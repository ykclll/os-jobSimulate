### 输出格式
每种算法打印一张表，按调度顺序排序，并且打印:
- 作业编号
- 到达时间
- 开始时间
- 周转时间
并在表后输出*平均周转时间*   
### 使用说明  
运行Main.c文件，根据提示输入txt文件名，系统可自动计算
txt文件格式: 每行代表一个作业，分别读入作业编号、到达时间、作业长度
